# React Tic Tac Toe Game
  
  -This is a tic tac toe game created following the React tutorial found [here](https://facebook.github.io/react/tutorial/tutorial.html)


## Installation

````
Clone repo

go to files and run in terminal:

npm install
npm start
````

## Requires

````
node
npm
````
